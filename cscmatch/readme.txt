The CSCMatch software requirements are available at https://csem.cc/static/csc205ab/CSCMatchSoftwareRequirements.pdf

The CSCMatch design requirements are available at https://csem.cc/static/csc205ab/CSCMatchDesignRequirements.pdf

The automated tests can be run locally as you develop your code. They can be found in the cscmatchtest folder.
For any step that has a test, make sure you run it and that it runs clean before advancing to the next step.
